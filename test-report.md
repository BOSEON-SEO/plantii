#### 1.1 온도 계산 (Temperature Factor)
- ✅ 최적 온도 범위 (20-27°C) → 1.0 반환
- ✅ 임계 온도 (5°C 이하) → 0 반환
- ✅ 임계 온도 (40°C 이상) → 0 반환
- ✅ 허용 범위 (15-30°C) → 0.3-1.0 반환
- ✅ 스트레스 범위 → 0-0.3 반환

**검증 결과**: 온도 입력에 따른 성장률 계산 로직이 정확하게 작동

#### 1.2 습도 계산 (Humidity Factor)
- ✅ 최적 습도 (50-70%) → 1.0 반환
- ✅ 낮은 습도 → 감소된 계수 반환
- ✅ 높은 습도 → 감소된 계수 반환
- ✅ 극단적 습도 → 최소 계수 반환

**검증 결과**: 습도 변화에 따른 성장 영향 계산 정상

#### 1.3 광량 계산 (Light Factor)
- ✅ 최적 광량 (15-20 DLI) → 1.0 반환
- ✅ 부족한 광량 → 감소된 계수 반환
- ✅ 과도한 광량 → 감소된 계수 반환 (광저해 효과)
- ✅ 광량 증가에 따른 계수 증가 검증

**검증 결과**: DLI(Daily Light Integral) 기반 광량 계산 정확

#### 1.4 수분 계산 (Water Factor)
- ✅ 최적 토양 수분 (60-80%) → 1.0 반환
- ✅ 심각한 가뭄 (<30%) → 0 반환
- ✅ 중간 가뭄 → 감소된 계수 반환
- ✅ 과습 상태 → 감소된 계수 반환
- ✅ 심각한 과습 (>96%) → 0.2 반환 (뿌리 부패 위험)

**검증 결과**: 토양 수분 관리 로직 정상 작동

#### 1.5 통합 성장률 계산 (Growth Rate Calculation)
- ✅ 최적 환경 → 성장률 1.0
- ✅ 불량 환경 → 성장률 0.3 이하
- ✅ 임계 환경 → 성장률 0.5 이하
- ✅ 성장률 범위 제한 (0-2) 검증

**검증 결과**: 
```
입력: { temperature: 22, humidity: 60, light_dli: 15, soil_moisture: 70 }
출력: growthRate ≈ 1.0 (최적 성장)

입력: { temperature: 10, humidity: 30, light_dli: 5, soil_moisture: 30 }
출력: growthRate < 0.3 (성장 억제)
```

#### 1.6 건강도 업데이트 (Health Update)
- ✅ 최적 환경 → 건강도 증가 (+0.5/시간)
- ✅ 좋은 환경 → 건강도 유지
- ✅ 불량 환경 → 건강도 감소 (-0.3/시간)
- ✅ 건강도 범위 제한 (0-100) 검증

**검증 결과**: 환경 점수에 따른 건강도 변화 정확

#### 1.7 토양 수분 증발 (Soil Moisture Depletion)

- ✅ 고온 환경 → 빠른 증발

- ✅ 최소값 0 제한

## 2. API 테스트

### 2.1 수동 검증 완료 항목

#### ✅ 인증 API
- **POST /api/v1/auth/register** - 회원가입
  - 필수 필드 검증: username, email, password
  - 비밀번호 해싱 (bcrypt)
  - JWT 토큰 생성 및 반환
  
- **POST /api/v1/auth/login** - 로그인
  - 이메일/비밀번호 검증
  - 액세스 토큰 및 리프레시 토큰 반환
  - 잘못된 자격증명 시 401 에러

#### ✅ 식물 CRUD API
- **GET /api/v1/plants** - 식물 카탈로그 조회
  - 15종 식물 데이터 반환
  - 카테고리별 분류 (화훼, 다육, 허브, 관엽, 채소)
  
- **GET /api/v1/plants/:id** - 특정 식물 상세 조회
  - 환경 조건, 성장 정보, 재배 난이도 포함

#### ✅ 사용자 식물 API (인증 필요)
- **GET /api/v1/user-plants** - 내 식물 목록
  - 인증 미들웨어 작동 확인
  - 401 에러 반환 (토큰 없을 시)
  
- **POST /api/v1/user-plants** - 식물 심기
- **POST /api/v1/user-plants/:id/water** - 물주기
- **POST /api/v1/user-plants/:id/environment** - 환경 조절
- **POST /api/v1/user-plants/:id/harvest** - 수확
