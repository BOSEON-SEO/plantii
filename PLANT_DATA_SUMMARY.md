# 식물 생육 데이터 요약

## 개요

실제 식물 14종의 생육 환경 데이터를 조사하여 정리했습니다. 각 식물은 시뮬레이션에 필요한 다음 정보를 포함합니다:

- 최적 온도, 습도, 광량
- 물 주기 및 토양 수분 요구
- 성장 속도 및 성숙 기간
- 시들기 조건 (회복 가능/치명적 수준)
- 3-5개의 상세 성장 단계

## 환경 변수 범위

### 온도 (°C)
- **저온 선호**: 상추(12-20), 시금치(10-18), 로즈마리(15-20)
- **중온 선호**: 바질(18-25), 민트(15-27), 페튜니아(15-25)
- **고온 선호**: 토마토(16-28), 고추(18-28), 선인장(10-30)

### 습도 (%)
- **저습도 내성**: 선인장(10-30), 알로에(20-40), 산세베리아(30-50)
- **중간 습도**: 로즈마리(30-50), 포토스(40-60), 민트(40-60)
- **고습도 선호**: 몬스테라(60-80), 고추(60-80), 토마토(50-70)

### 광량 PPFD (μmol/m²/s)
- **저광 내성**: 산세베리아(50-200), 포토스(75-200), 몬스테라(100-300)
- **중간 광량**: 상추(200-400), 시금치(200-350), 민트(300-500)
- **고광 요구**: 토마토(700-1000), 고추(700-1000), 선인장(800-1200), 해바라기(800-1200)

### 물 주기 (일)
- **잦은 관수**: 상추(1.5일), 토마토(1일), 페튜니아(1.5일)
- **보통 관수**: 바질(2.5일), 고추(2.5일), 시금치(2일)
- **드문 관수**: 알로에(17.5일), 선인장(21일), 산세베리아(21일)

## 성장 단계 패턴

모든 식물은 일반적으로 다음 패턴을 따릅니다:

1. **발아/정착 단계** - 높은 습도, 온화한 온도, 낮은 광량
2. **모종/초기 성장 단계** - 뿌리 발달, 첫 잎 형성
3. **영양 생장 단계** - 활발한 잎/줄기 성장, 최적 광량 필요
4. **성숙/전환 단계** - 생식 준비 또는 구조적 변화 (예: 몬스테라 구멍 형성)
5. **생식/유지 단계** - 개화, 결실, 또는 지속적 생산

## 시뮬레이션 활용 예시

### 성장 속도 계산
```
실제 성장 = 기본 성장 속도 × f(온도) × f(습도) × f(광량) × f(수분)
```

각 환경 계수 f(x)는 0.0-1.0 범위:
- 최적 범위 내: 1.0
- 최소/최대 경계: 0.5
- 범위 밖: 0.0-0.3 (스트레스) 또는 0.0 (성장 중단)

### 건강 상태 판정
```
if 수분 < critical_fc_threshold:
    상태 = "죽음/영구 손상"
elif 수분 < wilting_fc_threshold:
    상태 = "시들기 (회복 가능)"
elif 수분 < water_fc_min:
    상태 = "건조 스트레스"
elif 수분 > water_fc_max:
    상태 = "과습 스트레스" (특히 다육식물)
else:
    상태 = "건강"
```

### 단계 전환 로직
```
if 현재_일수 >= 단계_최대_일수 AND 환경_조건_적합:
    다음_단계로_전환()
elif 환경_조건_불량:
    단계_진행_지연()
elif 특수_조건_충족 (예: 상추 >21°C):
    조기_개화_또는_추대()
```
